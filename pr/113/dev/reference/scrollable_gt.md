# Add scrollability to listings

Add scrollability to listings

## Usage

``` r
scrollable_gt(gt_tbl, height = "300px", min_table_width = "1200px")
```

## Arguments

- gt_tbl:

  a `gt` object

- height:

  a string to denote pixel height

- min_table_width:

  a string to denote pixel width

## Value

A `gt` object
