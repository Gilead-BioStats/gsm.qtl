# gsm.qtl: Good Statistical Monitoring QTLs

The Good Statistical Monitoring or 'gsm' suite of R packages provides a
framework for statistical data monitoring. 'gsm.qtl' is an extension
that contains the additional functions, workflows, and document template
to generate reports for QTL.
