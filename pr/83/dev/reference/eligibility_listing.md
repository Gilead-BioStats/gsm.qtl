# Generate Eligibility Listing

Generate Eligibility Listing

## Usage

``` r
eligibility_listing(df, download = FALSE)
```

## Arguments

- df:

  A `data.frame` containing the participant level dataset with
  eligibility

- download:

  a logical to determine whether or not to return a gt object or a plain
  data.frame ready for download

## Value

A `gt` object or `data.frame`
